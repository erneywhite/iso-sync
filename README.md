# ISO Sync — File Hash and Update Manager

Набор PHP-скриптов для вычисления SHA256-хэшей локальных файлов, кэширования результатов, а также автоматической проверки и обновления локальных ISO-образов по контрольным суммам с официальных серверов.

---

## Структура проекта

```
iso-sync/
├── index.php                # Веб-интерфейс отображения файлов и хэшей
├── update_iso.php           # Проверка актуальности и загрузка ISO-образов
├── generate_all_hashes.php  # CLI-скрипт вычисления и кэширования SHA256
├── crontab                  # Примеры cron-задач
└── files/                   # Папка для хранения ISO-образов
    └── .hash_cache/         # Кэш SHA256-хэшей (автоматически)
```

---

## Содержание

- **index.php** — веб-интерфейс отображения файлов, их размеров, дат и SHA256-хэшей. Хэши отображаются с префиксом `sha256:` и являются кликабельными для копирования в буфер обмена.
- **generate_all_hashes.php** — CLI-скрипт для рекурсивного обхода папки `files`, вычисления SHA256 и кэширования в `.hash_cache`. Поддерживает автоматическую очистку устаревших кэш-файлов.
- **update_iso.php** — проверяет локальные файлы по официальным контрольным суммам (`SHA256SUMS`, `SHA256SUM`) и при обнаружении устаревшей версии скачивает актуальный образ. После завершения автоматически запускает `generate_all_hashes.php`.
- **crontab** — примеры cron-задач для автоматизации: ежечасный пересчёт хэшей, ежедневное обновление образов.

---

## Функциональность

- Высокопроизводительное вычисление SHA256 с кэшированием для экономии времени
- Веб-интерфейс с удобной визуализацией и копированием хэшей в один клик
- Гибкое сопоставление локальных имён файлов с официальными на серверах
- Поддержка `latest` для автоопределения актуальной версии образа (например, CentOS Stream)
- Прогресс-бар загрузки в консоли
- Обнаружение зависания загрузки (отсутствие прогресса > N секунд) с автоматическим прерыванием и повтором
- Несколько автоматических попыток загрузки с пропуском проблемного файла по исчерпанию ретраев
- Очистка кэша хэшей для удаления неактуальных записей
- Автоматизация через cron

## Использование

### Веб-интерфейс

Запустите `index.php` на веб-сервере, укажите папку с файлами и кэш — интерфейс отобразит данные с возможностью копирования хэшей.

### Вычисление всех хэшей

```bash
php generate_all_hashes.php
```

Вычисляет и кэширует SHA256 для всех файлов в папке `files` с автоматической очисткой устаревших записей в `.hash_cache`.

### Проверка и обновление образов

```bash
php update_iso.php
```

Проверяет файлы по официальным SHA256-суммам. При зависании загрузки автоматически прерывает и повторяет запрос до исчерпания ретраев; проблемный файл пропускается.

### Автоматизация через cron

Добавьте содержимое файла `crontab` в вашу crontab (`crontab -e`):

```cron
# Пересчёт хэшей каждый час
0 * * * *  php /path/to/generate_all_hashes.php > /dev/null

# Обновление ISO ежедневно в полночь (если generate_all_hashes запускается в конце update_iso)
0 0 * * *  php /path/to/update_iso.php > /dev/null

# Обновление ISO по субботам (если generate_all_hashes вынесен в отдельное задание)
0 0 * * 6  php /path/to/update_iso.php > /dev/null
```

> 💡 Замените `/path/to/` на реальный путь к скриптам.

## Конфигурация

### update_iso.php

В массиве `$filesToUpdate` укажите локальные имена, папки, URL и официальные имена образов. Доступные опции:

- `force_download_without_checksum` — разрешить загрузку даже без контрольных сумм.
- Таймаут отсутствия прогресса и максимальное число ретраев настраиваются внутри функции `downloadFile()`.

### generate_all_hashes.php

Задаётся корневая директория `files` и директория кэша `.hash_cache`. Поддерживается обработка поддиректорий.
